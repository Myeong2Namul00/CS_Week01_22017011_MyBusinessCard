씨샾 재밌네요.

VS는 안 써보고 VScode나 텍스트작업용으로 좀 써본게 다인데,

과목 수강으로 VS 환경 적응하려니까 참 힘드네요 ㅋㅋㅋ

<img width="469" height="266" alt="image" src="https://github.com/user-attachments/assets/4d92857c-6c8c-4a95-9cf8-12217613929f" />

실행하면 이런 모습입니다.

왼쪽에는 사진올리긴 민망해서 그림판으로 그린 귀여운 그림이 있고

오른쪽에는 간단하게 기능구현한 버튼들이 있습니다.

배경색 랜덤 변경 버튼을 구현하는 게 심화 과제 중 하나였는데요,

랜덤 변경은 뭔가 맘에 안 들어서 Form을 하나 더 만들어서 원하는대로 변경할 수 있게 했습니다.

색상을 변경하기 버튼을 클릭하면 나오는 새로운 Form에서 원하는 색상 클릭으로 색상을 변경할 수 있습니다.

<img width="196" height="186" alt="image" src="https://github.com/user-attachments/assets/fa4a641a-354c-47f3-96f7-28e5a76eb4d8" />

새로운 Form이 나오는 위치는 Form1 오른쪽에 생성되게 해서 찾기 쉽게 만들었습니다.

예제코드로 올라온 랜덤 변경 코드는 대신 배경색랜덤변경*40 버튼에 넣어뒀는데요,

클릭하면 배경색이 순식간에 40번 랜덤으로 바뀝니다. 한번 하긴 심심하니까요.

파스텔톤 좋아해서 밝은계열 파스텔톤으로 나오게 해뒀습니다.

사진변경은 심화과제인 간단한 사진변경 기능을 구현해두었습니다.

클릭하면 슬픈 그림으로 바뀌고, 다시 클릭하면 돌아옵니다.

<img width="279" height="149" alt="image" src="https://github.com/user-attachments/assets/3ce41825-d3d7-4cb0-91f8-68ce17eed13a" />

깃허브 버튼을 클릭하면 확인창이 나오고, 예를 클릭하면 깃허브로 이동되게 해뒀습니다.
